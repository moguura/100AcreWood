Activity 1

Please talk about the image using い-adjectives. Look at Example 1.
Please press the microphone button and say いいです（iidesu）.

e.g.1

いい → てんきは（ 　　　）

In Example 1, the weather in the image is nice, so the correct answer is
てんきは いいです (The weather is good).
Now, let's try Example 2.　
Press the microphone button and say よくないです（yokunaidesu）.

e.g.2

いい → てんきは（ 　　　）

For Example 2, the weather in the image looks bad, so we'd choose
てんきは よくないです (The weather is not good).
